Моделирование простой биржи.

Следующие допущения были использованы:
* Сопоставляются заявки только по полному совпадению цены и количества.
* Баланс клиентов не проверятеся на отрицательность
* Сделки с самим собой не обрабатываются

Входные данные берутся из фалов указанных в параметрах запуска, или по умолчанию из файлов orders.txt и clients.txt в текущей директории.

Результат работы записывается в файл result.txt в текущей директории, а так же выводится в stdout.

Для заупска с логированием дебажной информации, запускать jvm с параметром -DExchange.debug=true
